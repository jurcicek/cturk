Listen to the audio of each sentence and type the words that you hear. There is no strict rule on the captitalization of each word but please use capitalized letters for acronyms.

Several important notes to be considered during this process:

1. Word forms and punctuations

-----------------------------------------------------------------------

Do not use Arabia form to transcribe digits, use the text form such as one, two three, ...

Do not input any punctuations, such as comma or period, unless the words contain them anyway.

Some examples of the exceptions:

good-bye

Mr.

I'm

For incomplete words (partially pronounced words), put "-" before or after the completed part of the word, such as

incomp-

-eech

2. Empty sentence and silence

-----------------------------------------------------------------------

For sentences without any real speech, put the below label:

(sil)

3. Non-speech event

-----------------------------------------------------------------------

When there are non-speech events in the sentence, transcribe it by using a description within brackets. For example:

(laugh)

(cough)

(sniff)

4. Hesitation words

-----------------------------------------------------------------------

For the words without any meaning which only indicates hesitation, try to choose one from the below list before creating a new one:
